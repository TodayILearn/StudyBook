# JavaScript란?

- **JavaScript**는 웹 브라우저에서 동작하는 스크립트 언어로, 웹 페이지에 동적인 기능을 추가하는 데 사용
- ES6(ECMAScript 2015) 이후 버전부터는 많은 새로운 기능이 추가되어 더욱 강력하고 효율적인 프로그래밍이 가능해짐

## 주요 특징

- **동적 타이핑**: 변수의 타입을 선언할 필요가 없음
- **프로토타입 기반 객체지향**: 클래스 없이 객체를 생성하고 상속할 수 있음
- **함수형 프로그래밍**: 함수를 일급 객체로 취급
- **이벤트 기반 프로그래밍**: 사용자 상호작용에 반응할 수 있음

## Node.js에서 index.js 실행 방법

Node.js 환경에서 JavaScript 파일을 실행하려면 명령줄에서 다음 명령어를 입력

```bash
node index.js
```

- 위 명령어를 실행하면 `index.js` 파일의 내용이 실행
- Node.js가 설치되어 있어야 하며, 현재 디렉토리에 `index.js` 파일이 존재해야 함
